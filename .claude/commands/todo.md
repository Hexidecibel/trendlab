# Add Todo Item

Quickly capture ideas and tasks to todo.md.

## Usage

```
/todo <description>
```

## Instructions

1. The item to add: `$ARGUMENTS`

2. Read or create `/Users/chriscushman/local/src/trendlab/todo.md`

3. Add the item as a bullet point:
   - Format: `- [ ] <the text>`
   - Add to the end of the appropriate phase section, or to a new "Unplanned" section at the bottom if it doesn't fit an existing phase

4. Confirm what was added and which section it was placed in

## Notes

- This is for quick capture - don't overthink it
- Use `/plan` later to process these into detailed plans
- Use `/work` to implement from the plan
